Giới thiệu về App Engine

App Engine cho phép bạn tạo và lưu trữ các ứng dụng trên cùng một hệ thống cung cấp năng lượng cho các ứng dụng của Google. App Engine cung cấp sự phát triển và triển khai nhanh chóng; quản trị đơn giản, không cần lo lắng về phần cứng, bản vá hoặc bản sao lưu; và khả năng mở rộng dễ dàng.

Các bước thực hiện để deploy project (nodejs):

  • Bước 1: Chuẩn bị một project nodejs sẵn
  • Bước 2: Đưa project lên github
    • git init
    • git add .
    • git commit -m "Khởi tạo project"
    • git push origin --all
  • Bước 3: Clone project về cloud.
    • git clone "link"
  • Bước 4: Deploy app
    • gcloud app deploy
